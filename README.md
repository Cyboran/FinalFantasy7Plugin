# Final Fantasy 7 Remake/Rebirth UI
A simple Final Fantasy 7 Remake/Rebirth UI plugin for XIVLauncher (Final Fantasy 14)


### Features
* Uses the health/mana/drive bars from FF7:R for the player and also adds an EXP bar
